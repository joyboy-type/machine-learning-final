1. Dataset Description

2. Data Processing Pipeline

Our preprocessing pipeline cleans the data in 5 sequential steps, transforming "dirty" raw data (25 unique labels, mixed types, outliers) into clean, standardized numeric arrays ready for model training.

Step 1 Label Standardization

Leet-speak mapping (D3RMAS0N→DERMASON), case unification (upper), whitespace stripping. 25 raw labels → 7 clean classes. Corrects 171 leet-speak + ~400 case/space variants.

Step 2 Feature Type Repair

Solidity: "?"→NaN→float64. Compactness: strip " cm" suffix→float64. All 16 features enforced to numeric types.

Step 3 Missing Value Removal

Drop rows with any NaN. The missing rate is low (~5% in Perimeter, ~2.8% in Solidity), so deletion is safe and avoids interpolation bias.

Step 4 Deduplication

Remove 49 exact duplicate rows. Small impact but important for unbiased evaluation.

Step 5 Outlier Clipping

3×IQR clipping per feature: values beyond [Q1-3×IQR, Q3+3×IQR] are clipped to boundaries. Preserves samples while suppressing extreme values. Also fixes physically impossible negative Area values.

Feature Eng. Z-score Standardization

StandardScaler: all features to μ=0, σ=1. Essential for SVM (RBF) and MLP. Chosen over Min-Max for better outlier tolerance. Optional PCA via --pca flag (95% variance).

XGBoost — Beyond-Classroom Algorithm

XGBoost (eXtreme Gradient Boosting) was proposed by Tianqi Chen in 2016 and has since become the dominant algorithm in Kaggle competitions and industrial ML systems. Key innovations over traditional GBDT include:

  • Second-order Taylor approximation of the objective function for more accurate gradient estimation
  • Built-in regularization (L1 + L2) to control model complexity and prevent overfitting
  • Column subsampling at both tree and split levels for improved generalization
  • Parallelized feature computation using sorted block structure
  • Automatic handling of missing values via learned default split directions

Model Registry System

Adding a new algorithm requires only a decorated factory function — no changes to any other module:

    return CatBoostClassifier(iterations=200, depth=6, verbose=False), True

Quick Start

# Install dependencies
pip install -r requirements.txt

# One-click full pipeline
python run.py --all

# Run specific algorithms
python run.py --algorithm svm,xgboost

# Data analysis only
python run.py --analysis-only

# With PCA dimensionality reduction
python run.py --algorithm all --pca

# Show all options
python run.py --help
